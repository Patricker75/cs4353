## Frontend

### Specifications
- NodeJS v18.17.1

### Getting Started
1. Install NodeJS or [NVM](https://github.com/nvm-sh/nvm)
2. ```
   cd frontend
   npm install
   ```
3. ```
   cd ..
   make frontend
   
   # OR
   # In frontend directory
   npm run dev
   ```

### Start
To start frontend server:

```
# In root directory
make frontend
```
